# Obsidian 日程管理器 (Calendar Scheduler)

一个为 Obsidian 深度定制的功能丰富、视觉美观的日程管理插件。它将日程数据直接存储在本地 Markdown 文件中，确保数据的私有性与可移植性，同时提供丰富的可视化视图和数据统计功能。

---

## 🌟 核心特性

- **Markdown 驱动**：所有日程数据均以标准 Markdown 格式存储，即使不使用插件，也能直接阅读和编辑。
- **本地优先**：无需云端同步，数据完全掌控在自己手中，支持离线使用。
- **全方位视图**：提供主页、月视图、周视图、日视图四种维度，满足不同粒度的管理需求。
- **直观交互**：支持点击创建、实时更新，提供极致流畅的 UX 体验。

---

## 🖼️ 视图功能详述

### 1. 主页 (Home View) —— 个人仪表盘
主页是插件的核心，提供全天候的概览：
- **今日统计**：实时显示当前日期、星期、节假日信息，以及今日日程的总数和待完成数。
- **习惯追踪趋势图**：
  - **作息趋势**：以折线图形式展示“起床”与“睡觉”时间的波动情况。
  - **健康指数**：通过柱状图和折线图结合，展示“每日步数”与“手机使用时长”。
  - **多维切换**：右上角支持切换 **本周/本月/本年** 数据范围。
  - **数据录入**：点击标题旁的 **铅笔图标**，即可快速记录指定日期的习惯数据。
- **时间统计仪表盘**：
  - **总时长统计**：统计并显示当前年度、月度、周度的日程总时长。
  - **日程类型占比**：通过 **甜甜圈图 (Donut Chart)** 直观展示不同一级分类的时间分配。
  - **详细明细统计**：通过 **堆叠条形图 (Stacked Bar Chart)** 展示一级分类及其下属二级分类的具体时长。
  - **交互式图表**：点击图表下方的 **分类图例**，可以动态隐藏或显示对应的分类数据。

### 2. 月视图 (Month View) —— 宏观规划
- **全局视角**：以月历形式展示整月安排。
- **快速管理**：点击日期单元格可快速创建日程；直接在日期格内查看任务。
- **详情预览**：鼠标悬停在日程项上可显示包含时间、分类、备注的浮层详情。

### 3. 周视图 (Week View) —— 周间统筹
- **时间轴布局**：采用 7:00 - 24:00 的标准时间块布局，清晰展示一周的时间分配。
- **点击创建**：点击周视图顶部的 **日期表头**，可直接为该天创建新日程。
- **精细排程**：支持在时间轴上查看日程的起止时间、标签和备注信息。
![5fca30578aefad8c6229f9f313cefa64](https://github.com/user-attachments/assets/08c5754f-c63a-41de-929f-fe66119c3c48)

### 4. 日视图 (Day View) —— 专注执行
- **侧边栏常驻**：默认在侧边栏打开，作为工作时的任务清单。
- **专注当下**：仅展示今日任务，支持快速勾选完成。

## ⚙️ 插件设置

在插件设置页面，可以进行高度自定义：
- **存储路径**：自定义日程数据存放的文件夹名称（默认 `日程计划`）。
- **时间轴自定义**：设置视图中显示的开始小时、结束小时以及时间间隔（15/30/60分钟）。
- **分类与标签管理**：
  - **增删标签**：支持灵活添加或删除一级分类和二级分类。
  - **色彩管理**：为每个一级分类指定特定的颜色，该颜色将同步应用到所有视图和图表中。

## 📂 数据存储结构

插件会在指定的文件夹内按月生成 Markdown 文件，格式规范且易读：

```markdown
## 1月28日 周三
#### 习惯打卡与追踪
- 起床：07:30
- 步数：8000
- 睡觉：23:30
- 手机：4.5h

#### ToDo List
- [ ] 09:00-10:00 晨会 #工作/实习
- [x] 14:00-15:30 插件开发 #项目任务/开发
```

## 🛠️ 安装

1. 将 `main.js`、`manifest.json`、`styles.css` 放置于库文件夹 `.obsidian/plugins/calendar-scheduler/` 下。
2. 在 Obsidian 设置 -> 社区插件中启用。
